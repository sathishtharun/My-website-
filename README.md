# Simple Login/Signup Website

This is a **simple login and signup website** built using **HTML, CSS, and JavaScript**.  

## Features
* User can **signup** with username and password
* User can **login** using the credentials
* After login, user sees a **Welcome page** with self-introduction
*Frontend-only project (no backend/database)

## Files
* `index.html` - Home page with links to login/signup
* `signup.html` - Signup page
* `login.html` - Login page
* `welcome.html` - Welcome page after login
* `script.js` - JavaScript logic for signup/login
* `styles.css` - Styling

## How to Run
1. Download the repository or clone it
2. Open the folder in **VS Code** or any editor
3. Open `index.html` in **browser** to start
4. Signup → Login → Welcome page
